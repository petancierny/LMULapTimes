# LMU Lap Times App - Ideas

## Core Experience
- Personal lap log with date, car, setup notes, weather, and session type.
- Delta to target pace with a colored trend line for improvement over time.
- Favorite tracks and quick jump list.
- One-click copy of benchmark times to clipboard.
- “Your pace tier” card with percentile and a short coaching tip.

## Track Intelligence
- Sector targets derived from lap targets and track guide pacing hints.
- Corner difficulty rating and time-loss hotspots per track.
- Track conditions presets (rubbered, green, wet) that adjust targets.
- Track “risk zones” overlay notes (track limits, kerb traps).

## Car & Class Comparison
- Car selector with per-car target overlays and “best car at track” history.
- Class-to-class comparison slider (LMGT3 vs LMP2 etc).
- BoP note panel for patch version context.
- Simulated stint pace vs hotlap pace toggle.

## Track Guide Enhancements
- Turn cards that collapse/expand with braking/turn-in/apex visuals.
- Add a “Source needed” badge per turn with a quick input to attach a link.
- Turn numbering overlay on the layout image.
- “Watch reference” links (video timestamp) per turn.

## Layout & Visuals
- Zoom and pan on the layout with a reset button.
- Layout fit modes: “Fit width”, “Fit height”, “Max detail”.
- Small minimap inset for long tracks (Sarthe).
- Dark-on-light layout toggle for printing.

## Data & QA
- CSV import/export for personal laps.
- Data validator that flags inconsistent patches or classes.
- “Data freshness” card showing last update and changes since last load.

## Sharing & Social
- Shareable track card image export.
- “Compare with friend” overlay where you can paste a lap time.
- Public link to a filtered view (track + class + tier).

## Offline & Performance
- Service worker for offline access to guides/layouts.
- Lazy load guide content and layout images.
- Cache-busting for updated data.

## Accessibility
- High-contrast mode toggle.
- Larger font sizing for track guides.
- Keyboard shortcuts for track/class switching.

## Future Content
- LMU patch changelog view.
- Setup baseline library per track.
- Race strategy calculator (fuel, stints, tire windows).
